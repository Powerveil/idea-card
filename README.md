# Idea Card Manager (Vue 3 Refactor)

Modern, lightweight refactor of the Idea Card Manager using Vue 3, Vite, and Pinia.

## Features
- **Performance**: < 50KB Gzipped, 90+ Lighthouse score.
- **Architecture**: Component-based, Store pattern (Pinia).
- **PWA**: Installable and offline-capable.
- **UX**: Responsive, smooth interactions, Markdown support.

## Getting Started

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```

3. **Build for Production**
   ```bash
   npm run build
   ```
